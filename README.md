## 🚀 Highlighted Project
### ZwiftPower → Excel Updater
**Repo:** `zwiftpower-excel-updater`

Python tool that automates collecting rider data from ZwiftPower profiles and updates a team Excel file with:

- weight
- zFTP
- power data (15s / 1min / 5min / 20min)

The project was built to simplify team management for captains by replacing repetitive manual profile checks with an automated workflow.
