## 🚀 Highlighted Project

### ZwiftPower → Excel Updater
**Repo:** `zwiftpower-excel-updater`

Python tool that automates collecting rider data from ZwiftPower profiles and updates a team Excel file with:

- weight
- zFTP
- power data (15s / 1min / 5min / 20min)

The project was created to simplify team management for captains by replacing repetitive manual profile checks with an automated workflow.

### What this project shows
- practical Python automation
- working with Selenium and dynamic web pages
- Excel file processing with structured output
- configuration via `.env`
- project documentation and development roadmap
- thinking about future Docker-based packaging
